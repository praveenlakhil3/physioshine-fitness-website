## Quick Start
1. Open `index.html` in your browser
2. Or run: `start index.html` (Windows)

## Customization
- Replace placeholder images in `images/` folder:
  - `logo.png`
  - `hero.jpg`
  - `trainer1.jpg`, `trainer2.jpg`, `trainer3.jpg`
- Update contact info in HTML
- Modify colors in `css/style.css`

## Images Needed
Download free images from:
- Logo: Gym logo
- Hero: Fitness gym interior
- Trainers: Professional trainer photos
